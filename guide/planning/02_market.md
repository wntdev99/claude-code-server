### 4. SWOT 분석

```markdown
## SWOT 분석

┌─────────────────────┬─────────────────────┐
│      Strengths      │     Weaknesses      │
│      (강점)         │       (약점)         │
├─────────────────────┼─────────────────────┤
│ • [강점 1]          │ • [약점 1]          │
│ • [강점 2]          │ • [약점 2]          │
│ • [강점 3]          │ • [약점 3]          │
├─────────────────────┼─────────────────────┤
│    Opportunities    │       Threats       │
│       (기회)        │       (위협)         │
├─────────────────────┼─────────────────────┤
│ • [기회 1]          │ • [위협 1]          │
│ • [기회 2]          │ • [위협 2]          │
│ • [기회 3]          │ • [위협 3]          │
└─────────────────────┴─────────────────────┘

### SWOT 전략
- SO 전략 (강점-기회): [전략]
- WO 전략 (약점-기회): [전략]
- ST 전략 (강점-위협): [전략]
- WT 전략 (약점-위협): [전략]
```
