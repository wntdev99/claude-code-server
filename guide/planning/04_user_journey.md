## 다이어그램 가이드

문서에 다음과 같은 mermaid 다이어그램을 포함하여 시각적으로 표현하세요:

### 사용자 여정 다이어그램 (journey)
```
\`\`\`mermaid
journey
    title 사용자 여정: [시나리오명]
    section 탐색
      문제 인식: 3: 사용자
      해결책 검색: 2: 사용자
    section 사용
      앱 설치: 4: 사용자
      핵심 기능 사용: 5: 사용자
    section 정착
      일상 사용: 5: 사용자
      추천: 4: 사용자
\`\`\`
```

### To-Be 의사결정 흐름도 (flowchart)
```
\`\`\`mermaid
flowchart LR
    A[문제 발생] --> B{솔루션 사용?}
    B -->|예| C[빠른 해결]
    B -->|아니오| D[기존 방식]
    D --> E[시간 소요]
    C --> F[만족]
\`\`\`
```
