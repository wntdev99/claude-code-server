# 문서 작성 규칙

## 개요

Phase별 작업에서 생성하는 문서의 작성 규칙과 품질 기준을 설명합니다.

> **중요**: 모든 문서는 **500자 이상**이며 **플레이스홀더 금지**입니다.

## 기본 규칙

### 1. 최소 길이

```
✅ 최소 500자 이상
  - 한국어 기준 500자
  - 영어 기준 300단어

❌ 짧은 문서
  - 200자 문서
  - 간단한 개요만 있는 문서
```

```

  실제 내용 작성
  구체적인 설명
  완성된 문장
```

# 아이디어 정의

## 문제 정의

현재 개인 Todo 관리 앱들은 단순히 할 일을 기록하고 체크하는 기능에 그치고 있습니다.
사용자들은 매일 수십 개의 Todo를 작성하지만, 우선순위를 정하거나 시간을 효율적으로
배분하는 데 어려움을 겪고 있습니다. 특히 직장인들은 업무와 개인 일정을 분리하지 못해
중요한 업무를 놓치거나, 개인 시간을 제대로 활용하지 못하는 경우가 많습니다.

또한 기존 앱들은 사용자의 작업 패턴을 학습하지 못해, 매번 같은 종류의 일을 반복적으로
입력해야 하는 불편함이 있습니다. 예를 들어, "회의 준비", "보고서 작성" 같은 반복 업무에
대해 템플릿이나 자동 제안 기능이 없어 생산성이 떨어집니다.

## 솔루션 개요

본 앱은 AI 기반 Todo 관리 시스템으로, 사용자의 Todo를 자동으로 분류하고 우선순위를
제안합니다. OpenAI GPT-4를 활용하여 사용자가 입력한 Todo의 내용을 분석하고,
카테고리(업무, 개인, 쇼핑, 건강 등)를 자동 분류합니다.

또한 사용자의 과거 패턴을 학습하여, 비슷한 Todo에 대해 예상 소요 시간과 최적 수행 시간을
제안합니다. 예를 들어, 사용자가 "회의 준비"를 자주 입력한다면, 이를 인식하고 필요한
서브 태스크(자료 준비, 참석자 확인 등)를 자동으로 제안합니다.

캘린더 연동을 통해 사용자의 일정을 고려한 Todo 스케줄링도 제공하며, Slack, Email 등
다양한 플랫폼과 연동하여 외부에서 받은 요청을 자동으로 Todo로 변환합니다.

## 핵심 가치 제안

1. **시간 절약**: AI가 자동으로 Todo를 분류하고 우선순위를 제안하여, 사용자가 계획을
세우는 시간을 80% 단축합니다.

2. **생산성 향상**: 개인화된 추천과 스마트 스케줄링으로 하루 평균 2시간의 업무 시간을
확보합니다.

3. **스트레스 감소**: 모든 Todo를 한 곳에서 관리하고, AI가 최적의 순서를 제안하여
"무엇을 먼저 해야 할지" 고민하는 스트레스를 제거합니다.

## 타겟 사용자

**주요 타겟**: 25-40세 직장인

- **세그먼트 1**: IT 업계 종사자
  - 다수의 프로젝트를 동시에 관리
  - 기술에 친숙하며 생산성 도구 적극 활용
  - 월 5,000원 이하의 구독 서비스 지불 의향 있음

- **세그먼트 2**: 프리랜서 / 1인 사업자
  - 업무와 개인 일정 분리 필요
  - 시간 관리가 수입에 직접적 영향
  - 효율성 향상을 위한 도구 투자 적극적

- **세그먼트 3**: 학생 (대학생, 대학원생)
  - 과제, 시험, 프로젝트 등 다양한 일정 관리 필요
  - 무료 또는 저렴한 가격 선호
  - 모바일 우선 사용

## 차별화 요소

### 기존 Todo 앱과의 차이

**Todoist, Microsoft To Do 대비**:
- AI 기반 자동 분류 및 우선순위 제안
- 개인화된 시간 예측 및 스케줄링
- 작업 패턴 학습 및 템플릿 자동 생성

**Notion, Asana 대비**:
- 더 간단하고 빠른 UI (Todo 입력에 특화)
- AI 어시스턴트가 자동으로 정리
- 개인 사용자 중심 (팀 협업 기능 최소화)

### 독자적 기술

1. **Context-Aware Categorization**: 단순 키워드 매칭이 아닌, 문맥을 이해하는 AI 분류
2. **Predictive Scheduling**: 과거 데이터 기반 최적 수행 시간 예측
3. **Cross-Platform Integration**: Slack, Email, Calendar를 하나로 통합
